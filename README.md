## 📌 Project Overview

This project presents an interactive Sports Footwear Sales Analysis Dashboard built using Microsoft Excel.
The dashboard analyzes footwear sales performance across brands, countries, categories, sizes, and payment methods.

It helps businesses understand sales trends, customer preferences, and brand performance to support data-driven decision making.

## 🎯 Business Objective

The main objective of this dashboard is to analyze footwear sales data and answer important business questions such as:

What is the total revenue generated from footwear sales?

Which brands perform the best?

Which categories generate the highest revenue?

Which countries contribute most to sales?

What are the most popular footwear sizes?

What payment methods are customers using?
